# Assets

This folder contains app assets like icons, splash screens, and images.

## Required Assets

When preparing for production, add the following files:

- `icon.png` (1024x1024px) - App icon
- `splash.png` (1284x2778px) - Splash screen
- `adaptive-icon.png` (1024x1024px) - Android adaptive icon
- `favicon.png` (48x48px) - Web favicon

## Placeholder Files

For development, you can use placeholder files or generate them using online tools like:
- [App Icon Generator](https://appicon.co/)
- [Expo Icon Generator](https://www.appicon.build/)

The app will work with default Expo assets during development.
